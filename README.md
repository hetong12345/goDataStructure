# goDataStructure
go的基础数据结构的实现，包括动态数组、链表、堆、栈、集合、索引、二分搜索树、avl数、红黑树等

#array 数组
使用goDataStructure.CreateArray() 创建 需要传入数组可用空间 cap 
返回一个数组变量
可以执行Interface中的array定义的方法。
